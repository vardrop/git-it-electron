Installiere Git auf deinem Computer und richte deinen Namen und deine eMail ein.

Git

Git ist eine freie Offene Software (der quellcode ist öffentlich) geschreiben von Linus Torvalds, welcher ebenso den Kernel des Linux Betriebssystem geschrieben hat.

Git ist ein Programm welches die Veränderungen überwachen kann, auch bekannt als das Programmieren in einer Versionsverwaltung. Solltest du schon einmal Veränderungen in einer Textdatei überwacht haben, kennst du das Konzept bereits.

Git installieren

Wir empfehlen die Installation von Git auf deinem Computer durch das herunterladen von der GitHub Desktop Anwendung. Wir werden die Desktop Anwendung für Git-it nicht verwenden (wir lernen im Terminal!) aber es beinhaltet Git und ist die einfachste Möglichkeit, um Git auf allen Plattformen auf die selbe Art zu installieren.

  • Windows: Nutze die Git Shell als dein Terminal.
  • Mac: Öffne GitHub Desktop und von den Einstellungen, wähle command line tools install. Nutze die Terminal Anwendung als dein Terminal.

Hast du schon Git, oder bist dir nicht sicher? Tippe git --version in dein Terminal und wenn die Versions Nummer größer als 1.7.10 ist, hast du alles! Für weiter Informationen, besuche die Git Website.

Git Software

Die GitHub Desktop Anwendung kann viel, aber nicht alles was Git kann, daher ist es wichtig das Terminal zu lernen. Aber wenn das einmal gelernt ist, wirst du froh sein, die Desktop Anwendung zu haben, da sie die Informationen deiner Projekte grafisch und mehr wie die GitHub Website darstellt.

Git selbst ist nicht wie andere Programme auf deinem Computer. Du wirst wahrscheinlich kein Icon auf deinem Desktop sehen, aber Git wird immer verfügbar sein und du wirst es jederzeit entweder aus dem Terminal oder der Git Desktop Anwendung aufrufen können.

Git einrichten

Sobald GitHub Desktop (und somit Git) installiert ist, öffne dein Terminal. Du kannst überprüfen, dass Git wirklich dort ist, indem du eingibst:

git --version

Das wird dir die Version von Git die auf deinem Computer ist zurückgeben und ungefähr so aussehen:

git version 1.9.1

Als nächstes richt Git ein, damit es weiß wem deine Arbeit zuzuordnen ist:

Richte deinen Namen ein:

git config --global user.name "Dein Name"

Jetzt deine eMail:

git config --global user.email "deineemail@beispiel.de"

Du hast die erste Herausforderung bestanden! Klicke den 'Überprüfen' Button an, um das Bestehen der Herausforderung zu überprüfen.

Dollar Zeichen in Code Dokumentation

Dollar Zeichen $ werden in Programm-Dokumentationen oft benutzt, um anzuzeigen, dass die Zeile command line Code ist (siehe die Code Beispiele oben). Du tippst allerdings das Zeichen $ nicht ein, sondern nur das, was danach folgt.
